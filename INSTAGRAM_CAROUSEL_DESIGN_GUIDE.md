## Standard Carousel Structure (4-5 Slides)

### Slide 1: Title Slide
- Background: Dark navy gradient
- Large headline (60-80pt bold)
- Emoji relevant to topic
- Subtitle with value prop (20-30pt)
- Layout: Centered, lots of breathing room

### Slide 2-4: Content Slides
- Top section: Prompt number (Prompt #1, Prompt #2, etc.)
- Middle: The actual prompt text (readable size)
- Bottom: Quick tip or note
- Use neon accent color as border/highlight

### Slide 5: CTA Slide
- "Save these prompts"
- "@eitanadar.ai"
- "More real estate automation tips"
- Link/bio CTA
- Strong visual element (color burst, glow effect)

## Design Rules for All Carousels

**Typography:**
- Headlines: 60-80pt, bold, neon cyan
- Subheadings: 40-50pt, bold, white
- Body text: 28-32pt, regular, white
- Small text: 20-24pt, regular, light gray

**Color Usage:**
- Use neon cyan (#00FFFF) for headlines/emphasis
- Use neon pink (#FF006E) for accent borders/highlights
- Use neon green (#39FF14) for CTAs/important info
- Keep background dark (navy #0a0e27)
- Always high contrast for readability

**Layout:**
- Leave 60px margin on all sides
- Center align for balance
- Use whitespace generously
- One idea per slide
- Easy to read in 2-3 seconds

**Visual Elements:**
- Emoji relevant to content (top right or center)
- Simple border accent (neon color, 3-4px)
- Optional: subtle gradient background
- No photos/complex images (text-based design)

## Design Tools to Use

**Canva Pro** (easiest)
- Use 1080×1350 template
- Search "Instagram carousel"
- Use neon color palette
- Apply consistent fonts
- Export as JPG

**Figma** (most control)
- Create 1080×1350 frame
- Design once, duplicate for carousel
- Use hex color codes directly
- Create components for consistency

**Adobe Illustrator** (professional)
- Full design control
- Export optimized JPGs

---

## Quick Workflow

1. **Choose design tool** (Canva recommended for speed)
2. **Create title slide** with carousel number and emoji
3. **Copy structure** for prompt slides (same layout each time)
4. **Add each prompt** from the prompts document
5. **Create CTA slide** with @eitanadar.ai
6. **Export as JPG** at 1080×1350
7. **Save to folder:** `projects/instagram-carousels/[carousel_name]/images/`
8. **Post with:** `/instagram upload [carousel_name] [caption]`
