### 2. TYPOGRAPHY SYSTEM

#### Recommended Font Combinations

**For Luxury/Professional Aesthetic:**
- Headline Font: Playfair Display Bold (serif) or Montserrat Bold (sans-serif)
- Body Font: Lato or Roboto (clean sans-serif)
- Hierarchy: Title 4.0em / Subtitle 3.0em / Body 1.5em

**For Modern/Contemporary:**
- Headline Font: Montserrat Bold or Proxima Nova Bold
- Body Font: Roboto or Inter
- Hierarchy: Title 3.5em / Subtitle 2.5em / Body 1.25em

**For Warm/Approachable:**
- Headline Font: Playfair Display or Georgia (serif, warmer feel)
- Body Font: Proxima Nova or Avenir
- Hierarchy: Title 3.75em / Subtitle 2.75em / Body 1.4em

#### Key Typography Rules (Applied Across All Carousels)

1. **Font Limit**: Max 2-3 typefaces per carousel (1 headline + 1 body + optional accent)
2. **Sizing Hierarchy**: Use distinct size ratios to establish clear information hierarchy
3. **Minimum Font Size**: 24px for mobile viewing (audience scrolls on phones)
4. **Contrast**: High contrast between text and background (test: readable one-arm distance)
5. **Font Weight**: Bold/condensed for headlines (grabs attention, saves space)
6. **Spacing**: Extra padding around headlines so letterforms have room to breathe
7. **Words Per Slide**: Maximum 10-15 words per slide for optimal readability
8. **Line Length**: Shorter line length on mobile (consider text box width)

#### Serif vs Sans-Serif Rules
- **Serif (Playfair, Georgia, Garamond)**: Use for luxury brands, high-end properties, traditional elegance
- **Sans-Serif (Montserrat, Lato, Roboto)**: Use for modern brands, contemporary properties, younger demographics
- **Mixing**: Serif headline + sans-serif body = editorial, magazine-like, high-end feel

### 3. LAYOUT STRUCTURE

#### The 60/40 Rule (Most Critical Principle)
- **60%**: Empty space (breathing room, margins, padding)
- **40%**: Content (text, images, graphics)
- Result: Clean, professional appearance; reduces cognitive overload

#### Standard Slide Dimensions
- **Format**: Portrait orientation (1080 x 1350px or 4:5 aspect ratio)
- **Why**: More vertical real estate than square, feels less cramped
- **Safe Area**: Leave 50px margin on all sides for platform variations

#### Visual Hierarchy Placement (Top to Bottom)

```
[Top 10%] .................... Logo or branding element (optional)
[Top 25-35%] ................. Headline (main message, 5-8 words)
[Middle 35-55%] .............. Primary visual or graphic element
[Bottom 35-45%] .............. Supporting text or secondary info
[Bottom 10%] ................. CTA or next-slide indicator
```

#### Information Structure

**For Property Listing Slides:**
- Headline: Location/Property Type (e.g., "Lakefront Luxury. Chicago")
- Visual: High-quality property photo (70% of slide)
- Supporting: Key stat (price, beds, or unique feature)
- CTA: "Swipe for more →"

**For Educational/Tips Slides:**
- Headline: Benefit-driven title (e.g., "3 Things First-Time Buyers Miss")
- Visual: Icon, graphic, or illustrative element
- Body: One key point (bullet format, max 1 sentence)
- Design: Number badge, progress indicator

**For Story/Narrative Slides:**
- Headline: Hook or question
- Visual: Lifestyle photo or illustration
- Body: Relatable text (max 2 sentences)
- Design: Minimal text overlay on image

### 4. VISUAL ELEMENTS & GRAPHICS

#### Shapes That Perform Well

1. **Circles** — Used for: Emphasis, avatars, progress counters, luxury aesthetic
2. **Lines** — Used for: Dividers, visual flow guides, connecting elements, structure
3. **Rectangles/Boxes** — Used for: Text containers, photo frames, information sections
4. **Geometric Patterns** — Used for: Background texture, visual interest, brand distinctiveness
5. **Gradients** — Used for: Depth, modern aesthetic, background interest (subtle preferred)

#### Icons & Decorative Elements

- **Arrow Icons**: Placement (right edge, subtle) guides the next swipe
- **Badges/Numbers**: Progress indicators, step counters, ranking
- **Check Marks**: Confirmation, list items, completed steps
- **Highlight Shapes**: Circles or boxes behind key numbers/words for emphasis
- **Doodles**: Minimal, hand-drawn style adds personality without clutter
- **Texture/Patterns**: Subtle (20% opacity max) to avoid visual noise

#### Visual Flow Techniques

- **Spanning Elements**: Background shape, line, or character that moves across slides visually pulls viewers through content
- **Consistent Icon Set**: Use same icon style throughout (all line, all filled, all cute, all corporate)
- **Directional Cues**: Subtle arrows, color gradients leading right, or negative space pointing right
- **Color Bleeding**: Let background colors extend to edges for modern feel

### 5. BACKGROUND STYLES

#### Option 1: Solid Color Backgrounds
- Use: Primary or secondary brand color from palette
- Effect: Clean, professional, focuses attention on content
- Risk: Can feel flat if no texture added
- Recommendation: Use with subtle texture overlay (10-20% opacity)

#### Option 2: Gradient Backgrounds
- Direction: Vertical gradients preferred (top-left to bottom-right also works)
- Opacity: Keep subtle (avoid over-saturation)
- Suggestion: Dark color → 20% lighter shade of same hue
- Effect: Modern, depth, visual interest
- Risk: Can distract from content if too bold

#### Option 3: Photographic Backgrounds
- Approach: Blurred property photos with overlay (50-70% opacity solid color)
- Effect: Lifestyle, aspirational, contextual
- Risk: Text must be readable; high contrast required
- Recommendation: Add semi-transparent color overlay before placing text

#### Option 4: Textured Backgrounds
- Styles: Subtle paper texture, fabric weave, marble, concrete (opacity 10-15%)
- Effect: Sophistication, tactile feeling, less digital
- Risk: Can reduce readability; must be subtle
- Best Use: Luxury, high-end aesthetic

#### Option 5: Pattern Backgrounds
- Styles: Geometric patterns, grid, minimal line patterns (opacity 10-20%)
- Effect: Contemporary, distinctive, brand-specific
- Risk: Must coordinate with color palette; can overwhelm
- Recommendation: Use for accent slides only, not every slide

#### Most Effective in 2026
White or cream backgrounds (80% preference) for readability + minimal distraction. When color needed, solid + subtle texture beats complex patterns.

## Professional vs Amateur Signals

### What Makes a Carousel LOOK Professional (Algorithmic Trust)

1. **Consistency**: Same fonts, colors, spacing across all slides
2. **Contrast**: Text readable from arm's-length distance; high contrast ratios
3. **Whitespace**: At least 50% empty space (not cramped)
4. **Typography**: Maximum 3 fonts; clear size hierarchy; proper letter spacing
5. **Alignment**: All elements align to a grid; no floating/random placement
6. **Color Coordination**: 2-4 colors max; from a cohesive palette (not random)
7. **Image Quality**: High-resolution photos; consistent cropping/sizing
8. **Finishing Details**: Subtle shadows, rounded corners, consistent line weights
9. **Hierarchy**: Headline stands out; supporting info is secondary
10. **Polish**: No typos, spelling errors, or misaligned elements

### Common Amateur Mistakes (Warning Signs)

1. **Inconsistent fonts** across slides — looks unprofessional, jarring
2. **Too much text** — causes scroll-away before reading
3. **Low contrast** text — hard to read; people scroll by
4. **Cluttered design** — violates 60/40 rule; overwhelming
5. **Random colors** — no color system; chaotic aesthetic
6. **Small fonts** — forces zooming; signals mobile-unfriendly
7. **Blurry images** — damaged photos, compressed quality
8. **Misaligned elements** — looks sloppy, reduces trust
9. **Too many effects** — shadows, gradients, glows everywhere; cheap look
10. **No white space** — violates 60/40 rule; feels cramped and amateur

## First Slide Design (The Hook)

### The 2-3 Second Window

You have 2-3 seconds max to stop someone from scrolling. Average scroll speed: 3-4 posts per second. First slide must win the visual war before importance is given to a single word.

### The Three-Element Hook Formula

1. **Headline (5-8 words)**
   - Communicates clear content promise
   - Creates curiosity gap ("What will I learn?")
   - Examples: "3 Things Buyers Miss", "Inside $2M Lakefront Home", "First-Timer's Closing Checklist"
   - Font: Bold, large (3.5em+), high contrast color

2. **Visual Pattern Interrupt**
   - Stops scroll in under 50 milliseconds
   - Options: High-contrast color, expressive face, striking graphic, bold shape
   - Should be immediately recognizable as different from feed scroll
   - Examples: Gold accent line, number badge, lifestyle photo, emoji-like graphic

3. **Directional Cue (Subconscious Invitation to Swipe)**
   - Subtle arrow pointing right
   - Visual element extending toward right edge
   - Negative space directing toward corner
   - Text prompt: "Swipe for more →" (minimal, subtle)
   - Effect: Subconscious signal to continue

### First Slide Design Best Practices

- **Layout**: Visual interrupt on left/center, headline on top, subtle arrow on bottom right
- **Color**: Use accent color strategically for pattern interrupt
- **Contrast**: Ensure headline contrasts sharply with background
- **Negative Space**: Don't fill entire slide; let design breathe
- **Mobile-First**: Test appearance at arm's length, zoomed/full screen
- **Motion**: Consider subtle gradient or color shift to bottom-right (guides eye to arrow)

### Performance Data

Carousels with high "swipe-through rates" (70%+ of slides seen) are distributed to 3-5x more non-followers than low-swipe carousels. First slide is 100% responsible for this metric.

## Slide Flow & Consistency

### Building 3-5 Reusable Templates

Top performers build 3-5 layout templates and repeat them systematically:

**Template 1: Hero + Headline** (Slide 1, 3, 7 pattern)
- Large property photo or lifestyle image
- Bold headline overlay
- Minimal additional text

**Template 2: Text-Focused** (Educational, tips, data)
- Solid background color
- Number badge or icon (left)
- Headline (top)
- Supporting text (center)
- Consistent styling

**Template 3: Two-Column** (Comparison, before/after)
- Left side: Image or visual
- Right side: Text/information
- Clear divider line

**Template 4: Full-Bleed Image** (Showpiece slides)
- Property photo filling slide
- Minimal text overlay
- Single data point or call-to-action

**Template 5: Call-to-Action** (Final slide, booking/contact)
- Branding prominent
- Clear button or contact info
- Testimonial or trust element

### Consistency Rules

- **Same background style** on related content
- **Same accent color** for emphasis/highlights throughout carousel
- **Same icon style** if using icons
- **Same headline format** (e.g., always number at start: "3 Things...")
- **Same spacing** between elements slide-to-slide
- **Same body text treatment** (all bullet points, all full sentences, etc.)

Result: Recognizable series that builds trust and audience habit.

### Property Listing Carousels (Highest Engagement)
- Slide 1: Hero photo + location hook
- Slides 2-6: Photos (interior, exterior, details)
- Slide 7: Key stats (price, beds, baths, square feet)

### Educational/Tips Carousels (High Saves)
- Slide 1: Hook headline ("5 Mistakes Sellers Make")

- Slide 7: Bonus tip or summary
- Slide 8: Lead magnet or CTA
- **Design**: Information-graphic style, numbered progression
- **Colors**: Primary color + 1 accent for progression/numbers
- **Result**: Highly saveable, reshared frequently

- Slide 1: Trend/stat that surprises
- Slides 2-5: Data breakdowns (charts, comparison, narrative)
- Slides 6-7: "What this means for you" translation
- Slide 8: Position for follow/consultation
- **Design**: Data visualization-forward; clean, organized
- **Colors**: Muted palette + accent for key numbers
- **Result**: Positions agent as expert, drives follow requests

### Lifestyle/Story Carousels (Connection)
- Slide 1: Emotional hook image
- Slides 2-5: Story progression (photos, lifestyle)
- Slide 6: Tie to property/service
- Slide 7: Personal connection or relatability moment
- Slide 8: CTA tied to emotional theme
- **Design**: Photo-heavy; minimal graphic elements; authentic
- **Colors**: Warm palette; minimal text
- **Result**: High engagement, DM conversations

## Practical Implementation Recommendations

### For Carousel Generator

1. **Build 3 Core Palettes** (Luxury, Modern, Warm)
   - Define exact hex values for primary, secondary, accent, white, gray
   - Each palette includes 5-7 color variations for flexibility

2. **Create 5 Master Templates**
   - Hero + Headline, Text-Focused, Comparison, Full-Bleed, CTA
   - Each template includes: safe margins, font sizing, placement guides, spacing rules

3. **Establish Typography Defaults**
   - Pair 1: Playfair Display + Lato (luxury)
   - Pair 2: Montserrat + Roboto (modern)
   - Pair 3: Georgia + Proxima Nova (warm)
   - Define size hierarchy: Title (3.75-4em), Subtitle (2.5-3em), Body (1.25-1.5em)

4. **Create Asset Library**
   - Icons (20+ most common in real estate: home, key, location, dollar, chart, etc.)
   - Shapes (circles, lines, rectangles, geometric patterns)
   - Textures (10-15% opacity subtle backgrounds)
   - Dividers and separators

5. **Define Layout Grid**
   - Base unit: 12-column grid at 1080px width
   - Margin: 50px (left/right), 60px (top/bottom)
   - Gutter: 20px between elements
   - Enforces alignment, consistency

6. **Set Performance Standards**
   - Minimum contrast ratio: 4.5:1 for text readability
   - Minimum font size: 24px for body text
   - Maximum fonts per carousel: 3
   - Maximum colors: 5 (primary, secondary, accent, white, gray)
   - White space ratio: 60% space, 40% content
