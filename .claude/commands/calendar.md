# /calendar

Manage your time and schedule. View what's coming up, schedule calls with prospects, and block focus time.

Read `.claude/skills/calendar/SKILL.md` for full operations and guidelines, then follow the instructions there based on what the user asks for.

Supported operations:
- `/calendar view [timeframe]` — See your schedule (today, week, next 7, next 30)
- `/calendar schedule [prospect] [topic] [duration]` — Book a call with a prospect
- `/calendar block [hours] [type] [date]` — Block focus time for deep work
- `/calendar conflicts` — Check for scheduling issues
- `/calendar capacity` — See availability and workload balance
