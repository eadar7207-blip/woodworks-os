# /browser

Controls a real browser to automate any web task.

**Usage:** `/browser [url or task description]`

**Examples:**
- `/browser go to kling.ai and generate an image with this prompt: ...`
- `/browser fill out the contact form at example.com`
- `/browser upload eitan_noglasses.jpg to the image reference field on kling.ai`

Invokes the browser skill. Claude takes over the browser, does the task, and reports back with screenshots.
