# /video-editing

AI video production powered by claude-code-video-toolkit. From concept to finished MP4.

Read `.claude/skills/video-editing/SKILL.md` for full operations and setup, then follow the instructions based on what the user asks for.

Supported operations:
- `/video-editing new [concept]` — Create a new video project end-to-end
- `/video-editing reel [topic]` — Generate an Instagram Reel for @eitanadar.ai
- `/video-editing demo [automation]` — Build an agency demo video for a prospect
- `/video-editing voiceover [script]` — Generate AI narration only
- `/video-editing music [vibe]` — Generate background music track
- `/video-editing redub [file]` — Replace voice on an existing video
- `/video-editing image [prompt]` — Generate a visual asset (FLUX.2 or Ideogram4)
