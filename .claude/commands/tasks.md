# /tasks

Manage your to-do list and priorities. All tasks persist in the wiki.

Read `.claude/skills/tasks/SKILL.md` for full operations and guidelines, then follow the instructions there based on what the user asks for.

Supported operations:
- `/tasks add [title]` — Create a new task
- `/tasks update [task name] [status]` — Move task status (pending, in-progress, done)
- `/tasks view [filter]` — Show tasks (today, week, high, in-progress, [project], all)
- `/tasks dashboard` — Task summary and workload snapshot
