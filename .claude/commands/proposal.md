# /proposal

Turn automation specs into professional proposals. Generate scopes, timelines, and pricing.

Read `.claude/skills/proposal/SKILL.md` for full operations and guidelines, then follow the instructions there based on what the user asks for.

Supported operations:
- `/proposal [prospect name] [automation brief]` — Draft a proposal
- `/proposal pricing [complexity]` — Get pricing guidance (simple, standard, complex, enterprise)
- `/proposal template [type]` — Load a proposal template
