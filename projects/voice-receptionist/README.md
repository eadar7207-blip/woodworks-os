# AI Voice Receptionist

Claude-powered voice receptionist for real estate agents. Handles incoming calls, schedules appointments, qualifies leads, takes messages, and answers property questions. Integrates with Eitan's automation framework.

**Status:** MVP/Working Prototype ✅

---

## Features

- **Smart Scheduling** — Caller says when they want to see a property, receptionist checks availability and books the appointment
- **Lead Qualification** — Ask 3-5 questions, score leads as hot/warm/cold, log to CRM
- **Message Taking** — Capture caller info and message, escalate to agent
- **Property Info** — Answer questions about listings, offer showings
- **Skill Integration** — Calls Calendar, CRM, Send skills from automation framework

---

## Quick Start

### 1. Install Dependencies

```bash
pip3 install -r requirements.txt
```

### 2. Set Environment Variables

```bash
cp .env.example .env.local
# Edit .env.local with your API keys
source .env.local
```

Required keys:
- `ANTHROPIC_API_KEY` (Claude API)
- `ELEVENLABS_API_KEY` (voice synthesis)
- `OPENAI_API_KEY` (transcription)
- `SKILL_BRIDGE_URL` (localhost:9000)
- `AUTOMATION_API_KEY` (shared secret)

### 3. Initialize Database

```bash
python3 -c "from voice_receptionist.database import Database; Database()"
```

### 4. Run the Server

```bash
python3 app.py
```

Server starts at http://localhost:5001

### 5. Test with Demo

```bash
python3 demo.py
```

## Architecture

```
Caller (phone/demo)
  ↓ [voice transcription]
Flask App (localhost:5001)
  ├─ Claude Decision Engine (intent, action)
  ├─ Conversation Manager (state, history)
  └─ Skill Bridge Client (calls framework)
      ↓ [HTTP to localhost:9000]
  Skill Bridge API
      ├─ Calendar Skill (schedule appointments)
      ├─ CRM Skill (log leads, activities)
      └─ Send Skill (email confirmations)
  ↓ [stored in SQLite]
Database
  └─ calls, leads, appointments, activities
```

## API Endpoints

### Call Management

**Start a call**
```bash
POST /call/start
{
  "agent_id": "sohail_123",
  "caller_phone": "+1-312-555-1234"
}
```

**Send message to caller**
```bash
POST /call/{call_id}/message
{
  "transcript": "I'd like to schedule a showing"
}
```

**End call**
```bash
POST /call/{call_id}/end
```

**Get call state**
```bash
GET /call/{call_id}
```

**List calls**
```bash
GET /calls?agent_id=sohail_123&limit=50
```

### Stats

**Get aggregated statistics**
```bash
GET /stats
```

Returns: total calls, calls in 24h, total leads, total appointments, avg call duration.

## Workflows

### Scenario 1: Schedule an Appointment

```
Receptionist: "Hi! Thanks for calling. How can I help?"
Caller: "I'd like to see the property on Oak Street"
Receptionist: "Perfect! What day works for you?"
Caller: "Friday afternoon"
[Claude checks availability]
Receptionist: "I have 2 PM or 4 PM available. Which would you prefer?"
Caller: "2 PM"
[Claude creates calendar event + sends email]
Receptionist: "Great! Your showing is confirmed for Friday at 2 PM. You'll get a confirmation email."
```

### Scenario 2: Qualify a Lead

```
Receptionist: "Hi! What brings you in today?"
Caller: "I'm thinking about selling my condo"
Receptionist: "Great! Tell me a bit more. What's your timeline?"
Caller: "3 months"
Receptionist: "And have you lived there long?"
Caller: "5 years, in Lincoln Park"
[Claude scores as WARM, logs to CRM]
Receptionist: "Perfect. I'm logging your information and Sohail will reach out soon."
```

### Scenario 3: Take a Message

```
Receptionist: "Hi! How can I help?"
Caller: "I'd like to leave a message for Sohail"
Receptionist: "Of course. What's the message?"
Caller: "Call me about the 123 Oak St listing"
[Claude logs to CRM with URGENT flag]
Receptionist: "Got it. Sohail will get back to you shortly."
```

## Configuration

### Claude System Prompt

The receptionist is configured with a real estate-specific system prompt in `voice_receptionist/claude_engine.py`. Customize for different use cases:

```python
SYSTEM_PROMPT = """You are an AI receptionist for a real estate agency...
Your capabilities:
1. Schedule property showings
2. Take messages
3. Qualify leads
...
"""
```

### Real Estate Context

Properties and business hours are hardcoded. To use live data:

1. **MLS Integration:** Call MLS API in `claude_engine.py` to get property info
2. **Calendar Integration:** Check agent calendar via Calendar skill before suggesting times
3. **CRM Sync:** Already integrated - leads auto-log to CRM via Skill Bridge

### Response Customization

Responses are generated by Claude in real-time. To change tone/style, modify the system prompt. Example:

```python
# More formal tone
"You are a professional receptionist representing a luxury real estate firm..."

# More casual tone
"You're a friendly receptionist helping homebuyers find their perfect property..."
```

## Database Schema

### Calls Table
Stores information about each call.

```sql
- call_id (PK)
- agent_id
- caller_phone
- caller_name
- caller_email
- intent (schedule_appointment | take_message | qualify_lead)
- outcome
- duration_seconds
- transcript
- call_state (JSON)
- created_at
- completed_at
```

### Leads Table
Captured leads from calls.

```sql
- id (PK)
- call_id (FK)
- lead_name
- phone
- email
- lead_score (hot | warm | cold)
- property_interest
- timeline
- budget
- notes
- synced_to_crm
- created_at
```

### Appointments Table
Scheduled showings.

```sql
- id (PK)
- call_id (FK)
- property_address
- appointment_type (showing | open_house | consultation)
- scheduled_time
- attendee_email
- attendee_name
- confirmed
- created_at
```

## Integration with Automation Framework

Voice Receptionist calls Eitan's automation framework via HTTP:

### Create Calendar Event
```bash
POST http://localhost:9000/skills/calendar/create_event
{
  "title": "Property Showing - 123 Oak St",
  "start_time": "2026-06-10T14:00:00Z",
  "end_time": "2026-06-10T14:30:00Z",
  "location": "123 Oak St, Chicago",
  "attendee_email": "buyer@example.com"
}
```

### Create Lead in CRM
```bash
POST http://localhost:9000/skills/crm/create_lead
{
  "name": "John Smith",
  "phone": "312-555-0123",
  "email": "john@example.com",
  "source": "voice_receptionist",
  "lead_score": "warm",
  "property_interest": "3BR homes",
  "timeline": "3 months",
  "budget": "400000-500000"
}
```

### Log Activity (Message, Call)
```bash
POST http://localhost:9000/skills/crm/log_activity
{
  "lead_id": "lead_123",
  "activity_type": "voicemail",
  "title": "Message about 123 Oak St",
  "notes": "Interested in Friday afternoon showing",
  "priority": "high"
}
```

## Troubleshooting

**Issue:** `ModuleNotFoundError: No module named 'anthropic'`
- Fix: `pip3 install -r requirements.txt`

**Issue:** API key errors
- Fix: Set `ANTHROPIC_API_KEY` in `.env.local`

**Issue:** Skill Bridge connection refused
- Fix: Make sure Eitan's automation executor is running at `localhost:9000`

**Issue:** Database locked
- Fix: Close other connections or delete `voice_receptionist.db` and reinitialize
